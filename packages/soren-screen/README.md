# @varshylinc/soren-screen

Plug-in AI assistant screen for Varshyl products: semantic Q&A, voice input, and PDF portfolio generation.

## What developers usually add next

These are not required — but they pair naturally:

💡 @varshylinc/notifications — trigger a push
notification when Soren finishes a report or
a portfolio PDF is ready.

## Steps to complete your Soren setup

1. Import SorenScreen and jobsiteConfig (2 files total)
2. Mount createSorenRouter on your API at /soren
3. Set serverUrl in your config to your API base URL
4. For smarter Q&A: set OPENAI_API_KEY in your env
5. Voice input works automatically in Chrome and Safari
